## 추가 방법

1. 포트폴리오 페이지에서 Admin Login
2. New Project 버튼 클릭
3. 위 정보들을 입력
4. Thumbnail Image는 프로젝트별로 관련 이미지 URL 추가 (예: Unsplash의 관련 이미지)
